      Monadius for Windows with OpenGL

            presented by Hideyuki Tanaka & Takayuki Muranushi

  ２０周年を迎えたグラディウスに。
  ゲームが好きな人たち、
  プログラミングが好きな人たちに。
  関数型パワーアップ横スクロールシューティング。

○ インストール方法

  monadius.zipを任意のフォルダに解凍してください。
  その後、monadius.exeを実行してください。
  monadiusをプレイするには、お使いのコンピュータに
  OpenGLがインストールされている必要があります。

  ダミーの.mp3ファイルを適当な音楽ファイルで置き換えると
  BGMとして再生します。

○ 操作

  ・移動            カーソルキー
  ・ショット         z
  ・ミサイル         x
  ・パワーアップ     c
  ・決定/スキップ   スペース
  ・ゲーム放棄       g
  ・ゲーム終了       q,escape

○ 基本ルール

  超遅延戦闘機ビックバイパー(以下自機)を操作して、
  雑魚敵をやっつけ敵弾をかいくぐり、無限に続くステージを
  攻略してください。

  自機は敵の攻撃や、敵本体、地形に衝突すると破壊されてしまいます。
  自機がやられても一定の場所からコンティニューが可能ですが、
  その場合パワーアップをすべて解除された状態からのスタートとなります。


○  パワーアップ

  赤色の敵や、特定の敵の編隊は全滅させることでパワーアップカプセルを
  出します。これを一定数だけ集めてパワーアップボタンを押すことで、
  自機を強化することができます。

  SPEED
    自機の移動速度を上げます。

  MISSILE
    地上の敵を効率よく攻撃することのできるミサイルを装備します。

  DOUBLE
    通常ショットが右と右上の2方向に発射されるようになります。

  LASER
    通常ショットに代わり、自機の前方に発射され、
    貫通力の高いレーザーを装備します。

  OPTION
    自機の位置を遅延評価し、自機と同じ兵装で攻撃を行う
    分身を装備します。

  ？
    自機の前方を防御するシールドを上下に2枚同時に装備します。


○  そのほかの特徴

　Monadiusをプレイするたびにキー入力の履歴がリプレイとして
　全自動で保存され、あとで再生して見ることができます。
　リプレイの再現性は(たぶん)証明されています。

　ステージは一種類しかありませんが、同じステージを何周も
　するうちに難易度が上昇します。

  グラディウスを知らない人にも適度に易しい一周目。
  上級者の方には高次周で死ねば楽しい復活。
　最初のほうが楽でつまらんとか、ぁりぇなぃ世界を見てみたいと
　いう人は、タイトル画面でステージに対応する数字を入力すると
　ステージセレクトもできます。

　1周以上クリアーすればスタッフロールが出ます。
　2周してたらメッセージが若干変わります。
　何回コンティニューしていてもスタッフロールは出ますが、
　ステージセレクトを用いてクリアーせずに先の面へ進んだ場合、
　リプレイは保存されるもののスタッフロールを見ることはできません。

　ゲームを起動してタイトル画面のまま放置しておくとデモプレイが
　再生されますので、攻略の参考にしてください。
　

○  コマンドライン引数

  次のものを渡せます。

  -r
    リプレイを保存しないようにします。

  -f
    フルスクリーンでの実行を試みます。

  ファイル名
    渡されたファイル名の拡張子が.replayであった場合、
    リプレイを読み込んで再生しようとします。
    monadius.exeのアイコンにリプレイファイルを
    ドラッグドロップしてもリプレイを再生できます。
